# No9 Comment App

เว็บแอปจำลองหน้าโพสต์และระบบคอมเมนต์ สร้างด้วย **C# ASP.NET Core Razor Pages (.NET 10)**

## Features

* แสดงหน้าโพสต์พร้อมรูปภาพ
* มีช่องให้พิมพ์คอมเมนต์
* กด `Enter` เพื่อเพิ่มคอมเมนต์
* แสดงคอมเมนต์ใต้โพสต์
* แสดงชื่อผู้คอมเมนต์เป็น `Blend 285`


## How to Run

```bash
dotnet run
```


## หมายเหตุ

ข้อมูลคอมเมนต์ถูกเก็บไว้ใน Memory
เมื่อปิดโปรแกรม ข้อมูลจะหาย
